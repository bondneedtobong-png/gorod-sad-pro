# Медиа-ассеты

Единое место для реальных фото и видео. Пути подставляются в `lib/media.ts` —
код компонентов менять не нужно.

| Файл                     | Где используется            | Конфиг в `lib/media.ts`     |
|--------------------------|-----------------------------|-----------------------------|
| `hero.mp4` + `hero-poster.jpg` | Видео-фон hero        | `HERO_VIDEO`                |
| `cases/<кейс>-before.jpg`| Слайдер «Было/Стало» (до)   | `BEFORE_AFTER_CASES[].before` |
| `cases/<кейс>-after.jpg` | Слайдер «Было/Стало» (после)| `BEFORE_AFTER_CASES[].after`  |

Пока файлов нет — все компоненты показывают изящные плейсхолдеры
(анимированные SVG/градиентные сцены), сайт выглядит законченным.

Логотип лежит отдельно — `public/logo.svg`. Чтобы поставить точный вектор,
просто замените этот файл (или компонент `components/brand-mark.tsx`).
